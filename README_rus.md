# Планиметрия
В этом репозитории будут все классы простых геометрических фигур.

Под простыми геометрическими фигурами понимается:
* Треугольники;
* Параллелограммы (а также их частные случаи: квадраты, ромбы и прямоугольники);
* Окружности;
* Трапеции.
